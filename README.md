# egt_ancs
## compile<br>
$ git clone https://github.com/s887432/egt_ancs<br>
$ cd egt_ancs<br>
$ cd build_cortex<br>
$ make<br>
## usage<br>
egt_ancs UART_DEVICE<br>
where <br>
UART_DEVICE is tty console device which is located at /dev/tttySx<br>
example:<br>
&#35; egt_ancs /dev/ttyS2<br><br>
